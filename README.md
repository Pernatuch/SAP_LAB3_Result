# SAP_LAB3_Result
Программа — симулятор клеточного автомата с растениями. Сетка 8×10 клеток, каждая хранит состояние: влажность (сухо/влажно/вода) и растение. Вода распространяет влагу на соседей, влажные клетки сохнут. Растения трёх типов: картофель (любит влажное), болотное (любит воду), кактус (любит сухое).

Инструменты: лопата (удаляет растение/осушает), ведро (создаёт воду), посадка (сажает выбранное растение). При клике программа определяет координаты клетки, проверяет выбранный инструмент, вызывает соответствующий метод объекта клетки, пересчитывает влажность всех клеток, полностью перерисовывает сетку.

Рост происходит при нажатии "Вырастить все": каждое растение сравнивает влажность клетки со своими предпочтениями, становится выросшим или мёртвым. Визуализация: цвет клетки по влажности, эмодзи растения, прозрачность для мёртвых. Архитектура: объекты Cell с методами, Renderer для отрисовки DOM, GridManager для логики сетки, PlantManager как фабрика растений, EventHandlers для кликов.
